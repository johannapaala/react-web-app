# Kaikenkarvaiset Web App

## Kuvaus
React-sovellus lemmikkien hyvinvointiin. Käyttäjä voi tallentaa tietoja lemmikeistään, kirjata oireita päiväkirjaan ja hallita profiileja. Sovellus toimii selaimessa ja tallentaa tiedot localStorageen, joten tietokantaa ei tarvita.

Komponenttipohjainen rakenne mahdollistaa profiilien luomisen, muokkaamisen ja poistamisen sekä oireiden lisäämisen, poistamisen ja aikajärjestyksen hallinnan. Lomakkeiden validointi varmistaa, että kaikki kentät täytetään ennen lähettämistä.

## Tekninen toteutus
- React + TypeScript  
- Routing: react-router-dom  
- State-hallinta: useState ja useEffect  
- Data tallennetaan localStorageen  
- Komponentit: Valikko, Paasivu, Profiili, LuoProfiili, Muokkaa, Poista, Paivakirja, Lisaa, Poistaoire, Lomake, Lahetetty
